## Get started

 - clone repo : `git clone https://github.com/Cubillosxy/flask-network.git`
 - install docker: [Docker](https://www.docker.com/get-started)
 `docker-compose build .`
 `docker-compose up`
 
 - visit `http://localhost:5000/ `
